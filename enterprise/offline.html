Download your Gitcolony Enterprise's offline version


For the time being Gitcolony Enterprise can only be installed on Linux.

NOTE: You'll need admin permissions to run Gitcolony installation.



Check dependencies to run Gitcolony:

To run properly, Gitcolony Enterprise needs the following software to be installed at runtime:

Docker version 1.7 or later

https://docs.docker.com/installation/

Docker-compose 1.2 or later

https://docs.docker.com/compose/install/


Install Gitcolony Enterprise

  1. Download the file from http://releases.gitcolony.com/gitcolony-enterprise.sh

  2. Copy and move that file to the destination server.

  3. Run: 

    sudo sh gitcolony-enterprise.sh

  4. After the installation is done, follow the steps here: Installation Guide (Skipping the first step).


Upgrading

Every time you receive an email with a new release from Gitcolony team, in order to update your local instance you must:

  1. Download the file from https://releases.gitcolony.com/gitcolony-update.sh

  2. Copy and move that file to the destination server.

  3. Run: 

    sudo sh gitcolony-update.sh

  4. Enjoy all the new features :)
